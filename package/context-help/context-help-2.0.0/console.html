Управление через консольный интерфейс

Для настройки маршрутизатора через консольный интерфейс необходимо подключиться к нему по последовательному интерфейсу с помощью терминальной программы, либо по сети через Ethernet-порт по протоколу SSH (порт 22). Есть несколько программ, поддерживающих работу по протоколу SSH, к примеру, Putty для OS Windows и ssh для OS GNU/Linux. В качестве логина необходимо ввести root, пароль — 1234.

Замечание

Следует заметить, что изменения, внесённые в конфигурацию маршрутизатора через консольный интерфейс, будут заменены после перезагрузки параметрами, указанными в веб-интерфейсе.
