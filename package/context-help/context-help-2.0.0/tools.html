Утилиты

В веб-интерфейсе на странице System/tools доступны различные утилиты, которые используются для анализа работы устройства либо проверки работоспособности.

  • syslog — предоставляет доступ к логам, получаемым от утилиты logread;

  • dmesg — предоставляет доступ к логам, получаемым от утилиты dmesg;

  • ping — позволяет проверить доступность узла с помощью протокола icmp;

  • mtr — позволяет проверить качество связи между устройством и удалённым узлом.
