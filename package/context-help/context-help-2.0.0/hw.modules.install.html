Установка сменных интерфейсов

Важно

Для установки модулей можно использовать только слоты A, B и C маршрутизатора.

Рисунок 3.16. Передняя панель без сменных модулей


Установка модуля выполняется следующим образом:

  1. Выключить питание маршрутизатора.

  2. Снять с маршрутизатора заглушку, закрывающую слот.

  3. Установить модуль в слот.

  4. Зафиксировать его при помощи крепёжных винтов.

  5. Включить питание маршрутизатора и произвести настройку установленного модуля.
