MIB Browser

MIB Browser , разработка компании iReasoning, доступна в платной и бесплатной версии, различающихся лицензиями и функциональностью. Заявлена поддержка платформ Windows, Mac OS X и Linux.

Рисунок 11.3. MIB Browser


Достоинства:

  • Удобный интерфейс.

  • Табличный просмотр SNMP таблиц.

Недостатки:

  • Бесплатная версия не поддерживает SNMPv3, позволяет загружать дополнительно к имеющимся не более 5 MIB.

  • Возможно некорректное отображение SnmpAdminString для имён SHDSL профилей.
