Настройка режима набора номера

Настройка режима набора номера выполняется на странице Hardware/VoIP/Dial mode. На странице присутствует одна настройка — PSTN type, или тип телефонной станции. Этот параметр указывает, какой способ дозвона поддерживается телефонной станцией.
