Загрузка файлов

В некоторых случаях может потребоваться загрузить файл на устройство. К примеру, это может быть пакет, который требуется установить на устройстве. Для выполнения этой задачи можно воспользоваться загрузчиком файлов, доступном в веб-интерфейсе на странице System/Upload file.

Рисунок 5.10. Загрузка файла


  • Path — каталог на файловой системе устройства, в который будет загружен файл;

  • File — файл на локальном компьютере, который требуется загрузить.
