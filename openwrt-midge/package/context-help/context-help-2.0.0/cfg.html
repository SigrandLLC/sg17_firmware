Сохранение/восстановление конфигурации

Веб-интерфейс позволяет сохранять текущую, восстанавливать сохраненную или заводскую конфигурацию маршрутизатора.

Сохранение конфигурации выполняется на странице System/Configuration/Backup:

Рисунок 5.7. Сохранение конфигурации


При нажатии на кнопку Backup будет предложено сохранить файл конфигурации, который в последствии можно будет загрузить.

Важно

После восстановления конфигурации устройство будет автоматически перезагружено.

Восстановление конфигурации выполняется на странице System/Configuration/Restore:

Рисунок 5.8. Восстановление конфигурации


На вкладке System/Configuration/default можно восстановить заводскую конфигурацию нажатием на кнопку Restore default.
