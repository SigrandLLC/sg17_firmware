
! Research - планируемые команды


Опции:
 -r n		- рекурсивно на n уровней
 -c			- в конце выдать hdb_count=n - по числу выведенных полей
 -q			- заключить в двойные (") ковычки 
 -qq		- заключить в одинарные (') ковычки 
 -l			- префикс local
 -e			- префикс export
 -d         - debug, увеличить уровень дебага
 -D	c		- установить разделитель c, default - '_' 
				(-D/  - sys/iface/eth0/mac)


Команды:
  gcp - get chain pairs
      hdb gp sys_ifaces*=*
        sys_ifaces_eth0_mac=xx:xx:xx:xx:xx:xx
        sys_ifaces_eth0_ipaddr=10.1.1.1
        ...

  gp - get pairs
      hdb gp sys_ifaces*
        eth0_mac=xx:xx:xx:xx:xx:xx
        eth0_ipaddr=10.1.1.1
        ...

  gc - get chains
      hdb gc sys_ifaces*
        sys_ifaces_eth0_mac=xx:xx:xx:xx:xx:xx
        sys_ifaces_eth0_ipaddr=10.1.1.1
	
  gn - get names
      hdb gn sys_ifaces_eth0_*
        mac
        ipaddr
