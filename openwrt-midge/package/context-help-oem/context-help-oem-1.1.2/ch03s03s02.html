Настройка сетевых параметров

В данной версии ПО, установленного на мультиплексоре, нет возможности задавать сетевые параметры через веб-интерфейс настройки, поэтому эту часть конфигурации требуется выполнить вручную. В первую очередь, необходимо убедиться, что на странице настройки сетевого интерфейса Network/Interfaces/hdlc*/General параметры Enabled и Auto неактивны:

Рисунок 3.19. Настройка интерфейса

После внесения необходимых изменений, необходимо активировать консоль мультиплексора: либо подключившись к нему по последовательному порту, либо по сети по протоколу SSH.

Соединение Е1 имеет тип точка-точка. Для активация соединения необходимо выполнить следующую команду:

# ifconfig hdlc0 192.168.200.1 pointopoint 192.168.200.2

  • hdlc0 - сетевой интерфейс
  • 192.168.200.1 - IP-адрес соединения на стороне мультиплексора

Если соединение не установилось, то надо деактивировать/активировать сетевой интерфейс:

# ifconfig hdlc0 down

# ifconfig hdlc0 up

Для активация соединения после загрузки мультиплексора, необходимо выполнить следующие команды:

# echo "ifconfig hdlc0 192.168.200.1 pointopoint 192.168.200.2" >> /etc/init.d/S90my_hdlc

# echo "ifconfig hdlc0 down" >> /etc/init.d/S90my_hdlc

# echo "ifconfig hdlc0 up" >> /etc/init.d/S90my_hdlc

Созданный файл необходимо сделать исполняемым:

# chmod +x /etc/init.d/S90my_hdlc
