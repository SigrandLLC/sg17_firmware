Установка и статус соединения

Для соединения по SHDSL один мультиплексор работает в режиме мастера (master), второй - в режиме слейва (slave). Конфигурация SHDSL соединения зависит от модели мультиплексора и используемых модулей. Настройка интерфейса SHDSL выполняется на странице System/SHDSL/dsl*.

мультиплексор ГМ-2-MPR

Замечание

Драйвер для SHDSL мультиплексора ГМ-2-MPR называется MPR-SHDSL-VB, его имя отображается в заголовке конфигурационного окна.

мультиплексоры ГМ-2-MPR поддерживают два способа конфигурирования интерфейса SHDSL (MPR-SHDSL-VB) - local и preact.

  • local - конфигурация параметров линии связи задаётся на обоих мультиплексорах.

  • preact - конфигурация параметров линии связи задаётся на мастере, а слейв получает настройки при установлении соединения. На данный момент при использовании этого режима максимальная скорость ограничена до 2304 Кбит/с.

Важно

Метод конфигурирования должен совпадать на обоих мультиплексорах.

Рисунок 3.11. Конфигурация SHDSL (MPR-SHDSL-VB)

  • Rate - пропускная способность линии связи, Кбит/c.

  • Mode - режим: мастер (master), слейв (slave).

  • Coding - метод кодирования.

  • Config - способ конфигурирования.

  • Annex, CRC, Fill, Inversion - настройки относятся к кодированию и заполнению сигнала, используются для обеспечения совместимости оборудования.

мультиплексор ГМ-2-MPR

Замечание

Драйвер для SHDSL мультиплексора ГМ-2-MPR (и модулей MPR-SHDSL-VB*) называется MPR-SHDSL-VB, его имя отображается в заголовке конфигурационного окна.

Настройка скорости соединения, режима синхронизации и режима кодирования выполняется только на мастере, слейв получает эти настройки во время инициализации соединения. Окно конфигурирования показано ниже:

Рисунок 3.12. Конфигурация SHDSL (MPR-SHDSL-VB)

  • Mode - режим: мастер (master), слейв (slave).

  • Clock mode - режим синхронизации, по-умолчанию используется режим plesio (плезиохронный). Перед выбором режима sync (синхронного) следует удостовериться, что всё оборудование, находящееся на линии связи (регенераторы, мультиплексоры) поддерживают данный режим.

  • Rate - пропускная способность линии связи, Кбит/c.

  • Coding - метод кодирования.

  • Annex, CRC, Fill, Inversion - настройки относятся к кодированию и заполнению сигнала, используются для обеспечения совместимости оборудования.

  • Power - включить подачу питания для регенераторов, находящихся на линии.

Установка и статус соединения

После сохранения настроек через некоторый промежуток времени соединение будет установлено, о чем будут сигнализировать светодиоды на лицевой панели мультиплексора. Так же информацию о состоянии соединения можно просмотреть на вкладке статуса:

Рисунок 3.13. Информация о состоянии соединения SHDSL

Вкладка статуса для MPR-SHDSL-VB отображает более подробную информацию. Если соединение не установлено, выводится следующая информация:

Рисунок 3.14. Информация о неустановленном соединении SHDSL

  • Link state - состояние соединения: offline - соединение не установленно, online - соединение установленно.

  • Power balance - баланс питания относительно земли. В норме на одном проводе +120 В, на другом - 120 В. Если перекос больше 30 В или отсутствует земля - это означает несбалансированность питания, загорается светодиод unbalanced на модуле SHDSL. Значение balanced этого параметра соответствует нормальной работе, unbalanced - несбалансированности питания.

  • Power overload - перегрузка питания. Слишком большое число регенераторов на линии, настроенных на питание от данного мультиплексора, вызывает перегрузку питания. На модуле SHDSL загорается светодиод overload. Значение no overload этого параметра соответствует нормальной работе, overload - перегрузке питания.

При установленном соединении добавляется следующая информация:

Рисунок 3.15. Информация об установленном соединении SHDSL

  • Actual rate - скорость установленного соединения, Кбит/с.

  • Actual line code - метод кодирования.

  • Actual clock mode - режим синхронизации.

  • SNR margin - соотношение сигнал/шум в линии связи.

  • Loop attenuation - затухание сигнала в линии связи.

Замечание

Соединения SHDSL будет установлено, если интерфейс dsl* активен, т.е. для него настроены сетевые параметры (установлен флажок enabled и интерфейс активен, подробнее - см. следующий раздел).
